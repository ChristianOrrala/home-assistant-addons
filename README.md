# Home Assistant Add-ons Repository

[![Home Assistant Add-ons](https://img.shields.io/badge/Home%20Assistant-Add--ons-blue.svg)](https://home-assistant.io/)

A collection of add-ons for Home Assistant.

## Add-ons

### [Heimdall Dashboard](./heimdall)

An elegant application dashboard and launcher that allows you to organize and access all your web applications from a single page.

**Features:**
- Auto-updates to latest version
- Configurable HTTP/HTTPS ports
- Persistent configuration storage
- Multi-architecture support

## Installation

1. Open Home Assistant
2. Go to **Settings** → **Add-ons** → **Add-on Store**
3. Click ⋮ (menu) → **Repositories**
4. Add this repository URL:
   ```
   https://github.com/ChristianOrrala/home-assistant-addons
   ```
5. Find and install the desired add-on
